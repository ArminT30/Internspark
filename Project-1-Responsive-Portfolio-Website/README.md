# Responsive Portfolio Website

## Overview

This project is a responsive personal portfolio website developed as part of my InternSpark Frontend Development Internship. It showcases my professional profile, technical skills, education, projects, achievements, certifications, and contact information through a modern and responsive web interface.

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Clean and modern user interface
- About section
- Education section
- Skills section
- Projects showcase
- Achievements section
- Certificates section
- Contact section
- Smooth scrolling and interactive user experience

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
